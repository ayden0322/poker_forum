榮譽系統 · 規則定案

博客邦 榮譽系統 規則 v1

六項規則討論定案 · 榮耀憑證(戰績解鎖·買不到) 與 門面裝飾(可買可換) 兩層並行
1

賽季週期 · 三層時間軸

定案
  • 週榜（已有）：高頻日常，人人有機會，每週有新聞
  • 月賽季（新增）：一個月一季，產生「本季神算王」等冠軍，發冠軍限定裝飾
  • 名人堂（新增）：賽季冠軍與各項紀錄永久留名
2

三條榜的口徑

定案
榜口徑門檻/防刷
神算王（準度）本季勝率本季 ≥30 注、只計賠率 ≥1.5
獲利王（P幣淨利）本季結算淨利本季 ≥20 注、不限賠率（本金＋量＋技術軸）
人氣王（影響力）本季被「跟這單並實下」次數跟單者手機驗證、每人每單去重 ⚠需新建跟單功能
  • 三榜 → 三位月冠軍：本季神算王 / 獲利王 / 人氣王，不同打法各有寶座
3

冠軍歸屬 · 月末加冕、次月在位

定案
  • 每月底結算出三榜冠軍
  • 冠軍在位整個次月，全站帶著冠軍限定裝飾
  • 月界卸冕收回，限定裝飾移交新冠軍
  • 即時榜同時顯示「本月現任榜首」＝正在搶的下一頂
4

防刷紅線

定案
  • 通用：只計有盤口賽事（足球/MLB，NBA 不列）、只計已結算注單、最低注額門檻（數字待定）、只有手機驗證帳號能上榜/計影響力
  • 連勝：連續命中須 賠率≥1.5＋注額達門檻；void/延賽/和局 → 略過不中斷
  • 命中類：該注注額須達門檻（不能用 1 幣單宣稱封神）
  • 影響力：跟單者須手機驗證、每位跟單者每張預測單只計 1 次
5

三種榮耀本質 + 門檻表

定案
成就徽章

永久，曾達成即擁有。→ 榮耀圖鑑

紀錄保持

可被超越，全站唯一保持人。→ 名人堂（舊保持人留歷史）

賽季冠軍

月冠·次月在位。→ 限定裝飾＋名人堂歷代

類成就條件（建議值·上線校準）
賽季神算王/獲利王/人氣王各榜月末 #1；#2/#3 亞軍季軍（輕徽章·無限定裝飾）
連勝五連勝 / 十連勝 / 二十連勝連續命中（賠率≥1.5、注額達門檻、void 略過）
命中冷門獵人 / 一戰封神單注命中 賠率 ≥5 / ≥15
準度神準射手（可分 70/75/80%）單月準度 ≥70%（≥30 場）
影響力帶單百人 / 千人 / 萬人累計被跟單 100 / 1,000 / 10,000
資歷開站元老 / 週年開站首月加入 / 加入滿一年
6

買不到 vs 可買 · 兩層

定案
榮耀憑證（買不到·戰績解鎖）

賽季冠軍前三、冠軍限定裝飾、全部成就徽章、名人堂紀錄、戰績數字本身、戰績稱號（神算王/冷門獵人…）

門面裝飾（可買·G幣·絕不真錢）

頭像框樣式、卡片主題/配色、上傳頭像、可買風格稱號（純風格/搞笑）

  • 紅線：牽涉戰績/成就/賽季 → 一律 earned-only；門面只收 G 幣、絕不接真錢（純虛擬經濟）
⚠ 規則討論中浮出的兩個實作前置
  • 影響力榜需先建「公開預測單 + 跟這單」功能——目前 P幣競猜沒有。一期可先不上人氣王，只上神算王＋獲利王（現有戰績就能算）。
  • 最低注額門檻的實際數字，要依 P 幣經濟規模定，先佔位、上線校準。
